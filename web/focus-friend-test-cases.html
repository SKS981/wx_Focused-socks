Focus Friend - 专注助手 测试用例文档

1. 测试概述

本测试用例文档针对 Focus Friend 专注助手 Web 应用进行全面测试，包括番茄钟计时器、陪伴角色、装扮系统、广告变现模拟和数据存储等核心功能。

2. 功能测试用例

2.1 番茄钟计时器功能

TC-001 开始计时器

高优先级
测试步骤 预期结果
1. 打开应用 页面加载完成，显示初始状态
2. 点击"开始"按钮 计时器开始倒计时，按钮状态变为禁用
3. 观察计时器显示 时间正确减少，每秒更新一次

TC-002 暂停计时器

高优先级
测试步骤 预期结果
1. 开始计时器 计时器正常运行
2. 点击"暂停"按钮 计时器停止，按钮状态变为禁用
3. 观察计时器显示 时间保持不变

TC-003 重置计时器

高优先级
测试步骤 预期结果
1. 开始计时器并运行一段时间 计时器正常运行
2. 点击"重置"按钮 计时器重置为初始时间，所有按钮恢复初始状态

TC-004 工作时间设置

高优先级
测试步骤 预期结果

TC-005 休息时间设置

高优先级
测试步骤 预期结果

2. 点击其他区域确认输入 设置保存成功
3. 刷新页面 休息时间保持为10分钟

TC-006 番茄钟完成切换

高优先级
测试步骤 预期结果
1. 将工作时间设置为1分钟 设置成功
2. 开始计时器 计时器开始倒计时
3. 等待计时结束 自动切换到休息时间，完成的番茄数增加1
