# Focus Friend 代码文档

## 1. 代码结构

Focus Friend 是一个基于 Web 的游戏化陪伴式专注计时器应用，主要由以下部分组成：

### 1.1 核心文件
- `focus-friend.html`：主应用文件，包含所有 HTML、CSS 和 JavaScript 代码

### 1.2 功能模块
1. **数据存储模块**：使用 localStorage 进行本地数据存储
2. **计时器模块**：管理专注时间的计时功能
3. **3D 房间交互模块**：实现 3D 房间的旋转和交互
4. **拼豆系统模块**：管理拼豆板和图案选择
5. **商店和装扮模块**：管理物品购买和房间装修
6. **游戏化功能模块**：包括经验值、等级、任务等
7. **统计和目标模块**：记录和展示专注数据

## 2. 变量说明

### 2.1 数据存储变量
- `storage`：数据存储对象，提供 get 和 set 方法操作 localStorage
- `user`：用户数据，包含名称、袜子数量、专注时间等
- `items`：物品数据，包含家具、壁纸等
- `userItems`：用户拥有的物品
- `room`：房间数据，包含壁纸和家具布局
- `focusSessions`：专注记录
- `perlerPatterns`：拼豆图案数据
- `currentPattern`：当前选中的拼豆图案
- `beadProgress`：拼豆进度
- `goals`：专注目标数据

### 2.2 模块变量
- `timerModule`：计时器模块，包含以下属性：
  - `timer`：计时器实例
  - `remainingTime`：剩余时间
  - `focusTime`：专注时长
  - `isFocusing`：是否正在专注
  - `isPaused`：是否暂停
  - `beadAnimationInterval`：拼豆动画间隔
  - `currentBeadIndex`：当前拼豆索引

- `room3D`：3D 房间交互模块，包含以下属性：
  - `isDragging`：是否正在拖动
  - `startX`：开始拖动的 X 坐标
  - `startY`：开始拖动的 Y 坐标
  - `currentRotationX`：当前 X 轴旋转角度
  - `currentRotationY`：当前 Y 轴旋转角度

### 2.3 其他变量
- `isDragging`：3D 房间拖动状态
- `startX`：拖动开始的 X 坐标
- `startY`：拖动开始的 Y 坐标
- `currentRotationX`：3D 房间 X 轴旋转角度
- `currentRotationY`：3D 房间 Y 轴旋转角度

## 3. 功能模块说明

### 3.1 数据存储模块

```javascript
const storage = {
    get: (key, defaultValue = null) => {
        const value = localStorage.getItem(key);
        return value ? JSON.parse(value) : defaultValue;
    },
    set: (key, value) => {
        localStorage.setItem(key, JSON.stringify(value));
    }
};
```

### 3.2 计时器模块
```javascript
const timerModule = {
    timer: null,
    remainingTime: 25 * 60, // 默认25分钟
    focusTime: 25,
    isFocusing: false,
    isPaused: false,
    beadAnimationInterval: null,
    currentBeadIndex: 0,

    updateDisplay: function() {
        const minutes = Math.floor(this.remainingTime / 60);
        const seconds = this.remainingTime % 60;
        document.getElementById('timerDisplay').textContent = `${minutes.toString().padStart(2, '0')}:${seconds.toString().padStart(2, '0')}`;
    },

    start: function() {
        if (this.timer) clearInterval(this.timer);
        this.timer = setInterval(() => {
            this.remainingTime--;
            this.updateDisplay();
            
            // 更新拼豆进度动画
            updateBeadProgress();
            
            if (this.remainingTime <= 0) {
                completeFocus();
            }
        }, 1000);
    },

    reset: function() {
        if (this.timer) {
            clearInterval(this.timer);
            this.timer = null;
        }
        this.remainingTime = this.focusTime * 60;
        this.isFocusing = false;
        this.isPaused = false;
        this.updateDisplay();
    },

    pause: function() {
        this.isPaused = true;
        if (this.timer) {
            clearInterval(this.timer);
            this.timer = null;
        }
    },

    resume: function() {
        this.isPaused = false;
        this.start();
    }
};
```

### 3.3 3D 房间交互模块
```javascript
const room3D = {
    isDragging: false,
    startX: 0,
    startY: 0,
    currentRotationX: 25,
    currentRotationY: 0,

    init: function() {
        const roomContainer = document.querySelector('.house');
        const perlerRoom = document.querySelector('.perler-room');

        if (roomContainer && perlerRoom) {
            // 鼠标按下事件
            perlerRoom.addEventListener('mousedown', (e) => {
                this.isDragging = true;
                this.startX = e.clientX;
                this.startY = e.clientY;
            });

            // 鼠标移动事件
            document.addEventListener('mousemove', (e) => {
                if (!this.isDragging) return;

                const deltaX = e.clientX - this.startX;
                const deltaY = e.clientY - this.startY;

                this.currentRotationY += deltaX * 0.5;
                this.currentRotationX -= deltaY * 0.5;

                // 限制旋转角度
                this.currentRotationX = Math.max(-30, Math.min(60, this.currentRotationX));

                roomContainer.style.transform = `rotateX(${75 + this.currentRotationX}deg) rotateY(${45 + this.currentRotationY}deg) translateZ(-9vw)`;

                this.startX = e.clientX;
                this.startY = e.clientY;
            });

            // 鼠标释放事件
            document.addEventListener('mouseup', () => {
                this.isDragging = false;
            });

            // 触摸支持
            perlerRoom.addEventListener('touchstart', (e) => {
                this.isDragging = true;
                this.startX = e.touches[0].clientX;
                this.startY = e.touches[0].clientY;
            });

            document.addEventListener('touchmove', (e) => {
                if (!this.isDragging) return;

                const deltaX = e.touches[0].clientX - this.startX;
                const deltaY = e.touches[0].clientY - this.startY;

                this.currentRotationY += deltaX * 0.5;
                this.currentRotationX -= deltaY * 0.5;

                this.currentRotationX = Math.max(-30, Math.min(60, this.currentRotationX));

                roomContainer.style.transform = `rotateX(${75 + this.currentRotationX}deg) rotateY(${45 + this.currentRotationY}deg) translateZ(-9vw)`;

                this.startX = e.touches[0].clientX;
                this.startY = e.touches[0].clientY;
            });

            document.addEventListener('touchend', () => {
                this.isDragging = false;
            });
        }
    }
};
```

## 5. 交互流程

1. **专注流程**：
   - 设置专注时长
   - 点击开始按钮
   - 小土豆开始拼豆
   - 计时器倒计时
   - 拼豆板进度更新
   - 专注完成后获得奖励

2. **3D 房间交互**：
   - 鼠标或触摸拖动旋转房间
   - 查看房间的不同角度

3. **商店和装扮**：
   - 浏览商品
   - 购买物品
   - 装修房间
   - 更换壁纸

4. **游戏化功能**：
   - 完成每日任务
   - 获得经验值
   - 升级
   - 与小土豆互动

5. **统计和目标**：
   - 设置每日和每周目标
   - 查看专注统计数据
   - 查看专注历史

## 6. 技术特点

- **模块化设计**：将功能模块化为对象，提高代码可维护性
- **响应式设计**：适配不同屏幕尺寸
- **3D 交互**：使用 CSS 3D 变换创建真实的 3D 房间效果
- **游戏化元素**：积分系统、升级系统、任务系统
- **本地存储**：使用 localStorage 实现数据持久化
- **动画效果**：电视屏幕动画、灯光效果、角色动画

## 8. 注意事项

- 本应用使用 localStorage 存储数据，清除浏览器数据会导致数据丢失
- 3D 房间交互需要支持 CSS 3D 变换的现代浏览器
- 部分动画效果可能在移动设备上表现不同
- 应用使用了外部图片资源，需要网络连接
